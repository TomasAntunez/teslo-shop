# Teslo API

1. Clone project

2. Install dependencies
```
yarn install
```

3. Clone the ```.env.template``` file and rename them to ```.env```

4. Change environment variables

5. Run postgres database
```
docker compose up -d
```

6. Run application in development
```
yarn start:dev
```
